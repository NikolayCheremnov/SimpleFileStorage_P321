ЗАДАЧА:

Разработать http web-api для хранения файлов с использованием Postgres (S3, redis - опционально).

Обработчики:

POST /api/file/upload + body -> ID загруженного файла: загрузить файл на сервер, для загруженного файла сохраняется:
	- метаинформация о файле (имя, тип, дата загрузки и пр.)
	- сам файл

GET /api/file/{ID} -> json body: получить метаданные файла по ID
GET /api/file/{ID}/download -> file: скачать файл с сервера

Postgres: используется для хранения файла и его метаданных

Опционально:
S3: используется для хранения самого файла вместо Postgres
redis: используется для кэширования метаданных файла из Postgres
